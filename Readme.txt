以下为模板使用说明，烦请同学们仔细阅读。


1.《未名物理评论》专用 LaTeX 模板分为“中文”、“英文”两套，请同学们根据论文语言自行选择。

2.同学们只需编辑 essay 文件夹中的以下文件即可：

     -> info.tex  包含标题、作者信息、摘要、关键词（可选），编写格式请参考模板提供的样例；
     -> text.tex  包含正文部分；
     -> ref.bib   包含参考文献。

  编辑完成后使用 pdfLaTeX 编译 main.tex 即可生成提交版论文。

3.请将全部 LaTeX 代码和 pdf 文件打包发送至邮箱：pprgaojian@163.com 。

4.本刊中文模板使用 UTF-8 编码，请用正确编码方式打开以防止乱码。



最后，感谢大家对《未名物理评论》第二期的支持！欢迎大家积极投稿！
